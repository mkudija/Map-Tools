# Requirements

* python 3.5.2
* matplotlib 1.5.1
* pandas 0.18.1
* requests 2.11.1
* geopy 1.11.0
* basemap 1.0.7


# Installation
Clone this repo using `https://github.com/mkudija/Maps.git` and run [Map-tools.ipynb](https://github.com/mkudija/Aircraft-Spec-Charts/blob/master/Aircraft-Spec-Charts.ipynb)
